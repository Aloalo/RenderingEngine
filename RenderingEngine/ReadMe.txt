Installation:

Add glew32.dll to your system folder

Add following to include directories of the project that uses RenderinEngine:
	"..\OGLtools\glfw\include;..\OGLtools\glew\include;..\OGLtools;..\RenderingEngine;"
Add following to library directories:
	"..\OGLtools\glfw\lib;..\OGLtools\glew\lib"
